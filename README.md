# Discord Calendar Bot

This repository contains a Discord bot that connects to one or more Google Calendars and posts event reminders to a configured Discord channel.

## Project Tooling

This project uses [uv](https://docs.astral.sh/uv/getting-started/installation/) to manage Python, dependencies, and project commands.

Install uv from the official documentation:

```powershell
powershell -ExecutionPolicy ByPass -c "irm https://astral.sh/uv/install.ps1 | iex"
```
